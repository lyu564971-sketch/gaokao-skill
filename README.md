# 雪峰 Agent — AI 高考志愿顾问

> 不是套壳。内置 24 省官方录取数据库，真的会查数据、盘志愿。
> 方法论源自张雪峰八本志愿填报专著 + 61节专业视频课程（1500+分钟），把"张雪峰那套"塞进了AI。
> 浏览器打开即用，不需要命令行。

## 5 分钟跑起来

**Windows：**
```
1. 装 Python（python.org，勾选 Add to PATH）
2. 下载本项目 ZIP → 解压到桌面
3. 双击 启动.bat → 浏览器自动打开
4. 点右上角 API设置 → 填 DeepSeek Key → 保存测试
5. 打字提问
```

**Mac：**
```
1. 装 Python（python.org）
2. 下载本项目 ZIP → 解压到桌面
3. 终端进入文件夹 → python3 server.py
4. 双击 打开我.html → 浏览器打开
5. 填 DeepSeek Key → 打字提问
```

> 📖 **完全零基础？** → [Windows 小白教程](小白教程-零基础也能用.md) | [Mac 安装教程](Mac安装教程.md)

## 界面

浏览器打开后就是一个聊天窗口，跟微信用法一样：

- **左侧对话列表** — 多轮对话独立保存，新建/切换/删除，关了浏览器回来还在
- **顶部模式切换** — 报考（正经志愿分析）←→ 娱乐（张雪峰风格吐槽），一键切
- **右上角 API 设置** — 填 Key、换模型，浏览器本地记住，下次不用重填
- **🌓 深色模式** — 晚上用不刺眼
- **输入框** — 打字回车发送，Shift+回车换行

## 核心能力

| 功能 | 说明 |
|------|------|
| 🗄️ **本地数据库** | 24 省 × 2024-2025 年官方投档线，42 万条，位次精确匹配 |
| 🤖 **AI 智能提取** | 自动从你的描述中识别省份、位次、专业偏好（"一万三"→13000、"川籍"→四川） |
| 🎯 **专业过滤** | 说"要计算机"只推计算机相关。医学、法学、会计自动筛掉 |
| 🔍 **联网搜索** | 填了 Tavily Key 用 AI 搜索（更准）；不填用百度兜底 |
| 📋 **省份政策感知** | 浙江 80 志愿 / 山东 96 志愿 / 老高考 8-12 所…自动适配 |
| 💬 **上下文记忆** | 追问不用重复说，agent 记得你前面说过什么 |
| 🔒 **隐私安全** | 数据全在本地，Key 存浏览器，不上传任何服务器 |

## 数据管线

```
你: "浙江655分位次10500，想学计算机电子，不接受调剂"
         ↓
    AI 提取 → 省份:浙江  位次:10500  专业:计算机,电子
         ↓
    本地 DB 查询 → keyword=计算机,电子 → 只返回对口学校
         ↓
    联网搜索 → Tavily/Baidu 搜最新分数线、就业薪资、学校介绍
         ↓
    整合 → [DB数据 + 联网数据] → 发给 LLM
         ↓
    输出 → 冲稳保建议，每条标注数据来源（DB/联网）
```

## 常见问题

**数据准不准？** — 来自各省考试院官方投档线，多轮联网交叉验证。但录取数据每年变，最终以官方为准。

**怎么更新？** — 重新下载 ZIP 覆盖文件夹。

**要钱吗？** — 个人使用完全免费。AI 调用按量计费（DeepSeek 聊几百次才几毛钱）。

**支持手机吗？** — 目前支持 Windows 和 Mac 电脑。[Mac 安装教程](Mac安装教程.md)

**能换别的 AI 吗？** — 能，API 设置里改。

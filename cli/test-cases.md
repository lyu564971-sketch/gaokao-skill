## 案例 1：河南理科 560 / 普通家庭 / 想学金融

**输入**：
```json
{ "province":"河南", "score":560, "rank":50000, "familyBackground":"一般",
  "candidates":["金融学","电气工程","计算机科学"] }
```

**❌ baseline（无 Agentic Protocol，普通 AI 腔）**：
> 金融是个不错的专业，但竞争激烈，建议综合考虑院校档次和个人兴趣。财经类院校和综合类大学的金融专业各有优势，你可以根据自己的分数和兴趣来选择。行行出状元，关键看个人努力……

**问题**：和稀泥、无数据、用了禁忌词、没考虑家庭背景、第一句不给判断。

**✅ round1（走完 Step2 + CHECKPOINT）**：
> 🎯 **停。普通家庭别碰金融前端。**
> 你 560 在河南够不上 211 财经类，金融前端 target school 是清北复交人 + 头部财经 211（[🟡 36氪]）。普通一本金融毕业的中位数去向是银行柜员/保险销售，月薪 5-8k（[🟡 麦可思]）。
>
> 🏆 推荐方向：
> - 电气工程 — 进国网/南网系统，稳定有保障（[🟡]）
> - 计算机应用型校 — 10 年高薪积累 ≫ 金融一辈子低薪，但要扛 35 岁危机（[🟡]）
> - 定向师范带编 — 普通家庭风险最低的出路（[🟢 考试院]）
>
> 🛑 避坑：金融工程（需家庭资源）、纯金融（无 target school 没出路）

**改进点**：第一句 headline 判断、用了真实中位数数据、按家庭背景分流、无禁忌词。

## 案例 2：广东物理类 480 / 困难家庭 / 分数尴尬

**输入**：
```json
{ "province":"广东", "score":480, "familyBackground":"困难",
  "candidates":["工商管理","护理","定向师范","中外合作金融"] }
```

**❌ baseline**：
> 这个分数可以考虑应用型本科，工商管理就业面广，中外合作金融虽然贵但能开阔视野……

**问题**：给困难家庭推荐"野鸡中外合作"和空泛管理，违反铁律 2。

**✅ round1**：
> 🎯 **480 困难家庭，别碰中外合作和工商管理，那是烧钱。**
> 困难家庭试错成本为零，唯一原则是**包分配/快速变现**（[⚪ 框架推断]）。
>
> 🏆 强推：
> - 定向师范 — 带编上岗，阶层不滑坡（[🟢]）
> - 护理 — 就业率最高，可出国，起薪 4-7k（[🟡]）
>
> 🛑 避坑：中外合作金融（学费 5-10 万/年，困难家庭赌不起）、工商管理（无硬技能，替代性强）

## 案例 3：江苏文科 620 / 优越家庭 / 想学哲学

**输入**：
```json
{ "province":"江苏", "score":620, "familyBackground":"优越",
  "candidates":["哲学","法学","经济学"] }
```

**✅ round1（优越家庭可放宽实用性）**：
> 🎯 **620 优越家庭，你可以从容谈理想——但法学比哲学性价比高得多。**
> 优越家庭不用为生计试错，哲学陶冶情操没问题。但法学同样人文素养 + 还有考公/红圈所的出路，比哲学多一条退路（[🟡]）。经济学进可攻退可守。
>
> 🏆 推荐：法学（顶级政法院校）> 经济学 > 哲学
> 🛑 唯一避坑：别去"野鸡"中外合作，优越家庭也不交智商税。

**要点**：优越家庭放宽实用性限制（铁律 2 第二分支），但仍保持判断立场。
